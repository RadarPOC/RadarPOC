What is the Tech Radar?

The Tech Radar is a list of technologies, complemented by an assessment result, called ring assignment. We use four rings with the following semantics:

  • MAINSTREAM — Technologies that can be used in new projects and existing projects.
  • PROPOSED — Technologies that can be explored in new projects, but must be agreed as MAINSTREAM before going to production.
  • CONTAIN — Technologies that should not be used in new projects, but don't actively need to be removed from existing products.
  • AVOID — Technologies that should not be used in new projects, and should actively be removed from existing products.
